# 📊 Data Analysis Portfolio (5 Projects)

## 📌 Overview
This portfolio contains 5 real-world data analysis projects across multiple domains:
- Retail (Supermarket Sales)
- Education (Student Performance)
- Weather Analysis
- Healthcare (COVID Analysis)
- Finance (Stock Market)

Each project includes data cleaning, analysis, visualization, and business insights.

---

## 📁 Project Structure
- data/ → datasets
- notebooks/ → Jupyter notebooks
- src/ → Python scripts
- visualizations/ → charts
- reports/ → PDF reports
- docs/ → detailed documentation
